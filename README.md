## 💡 Introduction

<p>


<br>
使用 HTML5 播放 MP4 文件时，部分文件需要较长的缓冲时间才能播放。这是因为某些软件会将 MP4 文件的索引信息放在文件末尾。

本软件可以将索引信息放在文件开头，从而解决缓冲时间过长的问题。
